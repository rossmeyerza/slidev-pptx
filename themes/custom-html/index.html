Custom HTML Deck

The point

Normal workspace files

The source of truth is ordinary HTML, CSS, and JavaScript inside the deck folder.

  • No framework dev server
  • No markdown compiler
  • No HMR proxy
  • Fast authenticated preview

Agent-friendly

The agent can edit the actual deck surface.

Later, the model can update `index.html`, `style.css`, and assets directly rather than squeezing layout into Markdown.
