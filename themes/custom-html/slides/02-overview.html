Overview

Plain files, live preview

Each slide is one HTML file on a fixed 1280x720 canvas. Edits appear in the preview as soon as they are saved.

  • One file per slide under slides/
  • Order and titles in deck.json
  • Design tokens and layouts in theme.css
  • Pixel-perfect PPTX export
Presenter notes go in an aside like this one. They never render on the slide; they appear in presenter mode alongside the next-slide preview.
